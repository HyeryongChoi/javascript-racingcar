<h2 align="middle">level1 - 자동차 경주 게임</h2>
<p align="middle">자바스크립트로 구현 하는 자동차 경주 게임</p>

## ⚙️ 실행가이드

다음 명령어를 입력해 패키지를 설치한다.

```
npm install
```

설치가 완료되었다면, 다음 명령어를 입력해 게임을 실행한다.

```
npm start
```
