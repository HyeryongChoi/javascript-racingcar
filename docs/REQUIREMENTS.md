# 👍 자동차 경주

## 🐾 기능목록

- [ ] 자동차 이름을 입력받는 기능
  - [ ] 자동차 이름들 입력값 파싱
  - [ ] 자동차 수 저장하기
  - [ ] 자동차 이름 입력값 공백 제거
  - (예외) 이름은 1 ~ 5글자 이하
  - (예외) 입력값이 없는 경우 예외처리
  - (예외) 자동차 이름이 중복되는 경우

- [ ] 시도 할 횟수 입력받기
  - (예외) 입력값이 숫자가 아닌 경우
  - (예외) 입력값이 0 이하인 경우

- [ ] 0 ~ 9 랜덤 값 구하는 기능

- [ ] 자동차 이동 여부 판단 기능

- [ ] 이동 결과 상태 저장 기능

- [ ] 이동 결과 출력 기능

- [ ] 우승자 선별 기능

- [ ] 우승자 출력 기능
